# Algorithm Register

## Legal Basis

Wet open overheid (Woo) - Article 2.1. Mandatory for government algorithm use.

## Registration Steps

1. Inventory all algorithms
2. Collect required information
3. Submit via register.algoritmes.overheid.nl
4. Publish in public register
5. Update on changes
