# NORA Architecture

NORA is the Dutch government reference architecture framework.

## Core Principles

| Principle | Description |
|-----------|-------------|
| User-centric | Services designed around citizen needs |
| Reuse before build | Prefer existing solutions |
| Data at source | Single source of truth |
| Open standards | Use open, documented standards |
| Privacy by Design | Privacy embedded from start |
| Security by Design | Security as foundation |

## EU Mapping

| NORA | EIF |
|------|-----|
| Reuse | Interoperability layer |
| Open standards | Technical interoperability |
| Privacy | Legal interoperability |
